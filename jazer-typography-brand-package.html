Do's and Don'ts

Best practices for maintaining brand consistency

‚úÖ DO

  • ‚Ä¢ Use Nunito at 800-900 weights for brand impact
  • ‚Ä¢ Apply gradient effects to Nunito headlines
  • ‚Ä¢ Use Outfit for all headers and navigation
  • ‚Ä¢ Use DM Sans for all body copy and forms
  • ‚Ä¢ Maintain consistent letter spacing
  • ‚Ä¢ Use uppercase for Nunito display text
  • ‚Ä¢ Ensure proper contrast on dark backgrounds
  • ‚Ä¢ Keep Nunito for short, impactful statements

‚ùå DON'T

  • ‚Ä¢ Use Nunito for body text or long paragraphs
  • ‚Ä¢ Mix more than 3 font families
  • ‚Ä¢ Use thin weights (below 600) for Nunito headlines
  • ‚Ä¢ Stretch, squish, or distort any typography
  • ‚Ä¢ Use Nunito for more than one sentence at a time
  • ‚Ä¢ Apply gradient to body text
  • ‚Ä¢ Use all-caps for body copy
  • ‚Ä¢ Ignore responsive font sizing

Using Clamp for Fluid Typography

All heading classes use clamp() for smooth scaling:

/* Fluid scaling between min and max */
.jazer-h1 {
  font-size: clamp(2.5rem, 6vw, 4.5rem);
  /* Min: 40px, Scales: 6% of viewport, Max: 72px */
}

.jazer-h2 {
  font-size: clamp(2rem, 5vw, 3.5rem);
  /* Min: 32px, Scales: 5% of viewport, Max: 56px */
}

Mobile-Specific Adjustments

@media (max-width: 768px) {
  /* Reduce letter spacing on mobile */
  .jazer-hero,
  .jazer-display {
    letter-spacing: var(--tracking-normal);
  }
  
  /* Increase line height for better readability */
  .jazer-body {
    line-height: var(--leading-relaxed);
  }
  
  /* Adjust stat sizing */
  .jazer-stat-number {
    font-size: clamp(2rem, 12vw, 4rem);
  }
}
